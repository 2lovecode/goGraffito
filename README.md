### GO语言实现的一些功能

#### 一.用法
  1. 克隆代码
        - git clone git@github.com:2lovecode/graffito.git
  2. 进入项目根目录
        - cd graffito
  3. 构建
        - go build
  4. 构建完成后，会在当前目录生成可执行文件
        windows: graffito.exe
        unix:graffito
  5. 运行
        - gra
